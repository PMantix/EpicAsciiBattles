# Open Questions (Resolved)

Decisions captured (2025-12-27):

1. Gore intensity levels: **Tame / Normal / Grotesque**.
2. Combat log verbosity: **Brief / Normal / Verbose** (user-adjustable).
3. Battle pacing guardrails: **yes** — implement an escalating fatigue soft-cap; fatigue accumulation accelerates in long battles.
4. Arena MVP terrain/features: **rivers, lava, destructible boulders, walls**.
5. “Standout” individuals rarity: **~1 in 1000**.
6. Intel/hints: only if unlocked **before** choosing; once battle begins, no hidden info (still conveyed via events/log, not explicit stats).
7. Content scope: primarily animals + fantasy creatures; occasional generic humans (knights/peasants). Generic descriptors only; no named real-world individuals or protected character names.
8. Data format: YAML + schema validation is acceptable.
