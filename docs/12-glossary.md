# Glossary

- **Arena**: the grid battlefield.
- **Cell**: one tile in the grid.
- **Combatant / Actor**: a single unit in the battle.
- **Species**: a template definition for a combatant (chicken, baboon).
- **Part**: an anatomical component (wing, beak, claw).
- **Tag**: a semantic label on parts/species used to derive behavior.
- **Tick**: one deterministic simulation step.
- **Event stream**: ordered events emitted by the sim for rendering/VFX.
- **Run**: a sequence of rounds until the player guesses wrong.
- **Round Offer**: the matchup selection screen.
